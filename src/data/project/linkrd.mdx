Linkrd is similar to the Linktree platform in providing a single page that serves as a central access point for various links found in someone's or a business's online presence. This page is designed to help users easily navigate through a variety of online resources they have, such as personal websites, social media profiles, online stores, blogs, and more, all in one organized and easily accessible place. With a clean and intuitive interface, Cataliank allows individuals and organizations to arrange their links in an engaging and efficient manner, enhancing user engagement and making it easier for others to discover and connect with their content across the web.

### Features

- Google oauth
- Logout
- Update Profile Image
- Update Profile Title
- Update Profile Bio
- Add Link
- Edit Link
- Remove Link
- Active Link
- Reorder Link
- Choose Theme
- Custom Theme
- Choose Background Type
- Choose Button Type
- Custom Font
- Add Social Icon
